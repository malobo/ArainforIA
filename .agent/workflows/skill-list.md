# Skill List

List all available skills with their triggers and categories.

## Steps

1. Read the file `.skills/INDEX.md` to get the complete catalog of skills
2. Present the skills organized by category (Core, Domain, Workflows)
3. For each skill, show:
   - Name and link
   - Triggers (keywords that activate it)
   - Priority level if marked

## Output Format

```
## Skills Disponibles

### Core
- skill-name: description (triggers: keyword1, keyword2)

### Domain  
- skill-name: description (triggers: keyword1, keyword2)

### Workflows
- skill-name: description (triggers: keyword1, keyword2)
```
